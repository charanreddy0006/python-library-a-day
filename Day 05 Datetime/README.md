# Day 05 - Datetime Module

## 📌 Overview

On Day 05, I explored Python’s built-in **datetime module**, which is used to work with dates and time.

It is widely used in real-world applications like scheduling, logging, and time tracking.

---

## 🧠 Topics Covered

* Current date and time
* Creating custom date and time
* Formatting dates
* Adding days using timedelta
* Finding difference between dates
* Countdown example

---

## 💻 Code Explanation

### 1. Current Date & Time

```python
datetime.now()
```

### 2. Formatting

```python
strftime("%d-%m-%Y")
```

### 3. Date Difference

```python
date2 - date1
```
