# Day 01 — NumPy

## What I understood
A library is ready-made code you import and use directly.
NumPy is the core library for numerical operations in Python.

## Key things
- np.array() is faster than a regular Python list for math
- .mean() .min() .max() .std() work on the whole array at once
- 2D arrays work like a table — rows and columns
- np.random.seed(42) makes random results reproducible

## Where I'll use this
- Car price prediction — all features are NumPy arrays internally
- Data analysis — quick stats on any column
