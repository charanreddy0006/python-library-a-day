# Day 08 - JSON Module

## 📌 Overview

On Day 08, I explored Python’s built-in **json module**, which is used for storing and exchanging data.

JSON (JavaScript Object Notation) is widely used in APIs and web applications.

---

## 🧠 Topics Covered

* Python to JSON conversion (dumps)
* JSON to Python conversion (loads)
* Writing JSON to file
* Reading JSON from file
* Updating JSON data

---

## 💻 Code Explanation

### 1. Convert Python → JSON

```python
json.dumps(data)
```

### 2. Convert JSON → Python

```python
json.loads(json_data)
```

### 3. File Handling

```python
json.dump(data, file)
json.load(file)
```
