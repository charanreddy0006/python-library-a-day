# Day 03 - Matplotlib Basics

## 📌 Overview

On Day 03, I explored **Matplotlib**, a powerful Python library used for data visualization.

It helps in creating graphs and plots to better understand data.

---

## 🧠 Topics Covered

* Line Plot
* Bar Chart
* Pie Chart
* Scatter Plot
* Labels and Titles

---

## 💻 Code Explanation

### 1. Line Plot

Used to show trends over data.

```python
plt.plot(x, y)
```

### 2. Bar Chart

Used for comparing categories.

```python
plt.bar(x, y)
```

### 3. Pie Chart

Used for showing proportions.

```python
plt.pie(data)
```

### 4. Scatter Plot

Used to show relationship between variables.

```python
plt.scatter(x, y)
```
