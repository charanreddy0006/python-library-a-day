# Day 14 - Functools Module

## 📌 Overview

On Day 14, I explored Python’s **functools module**, which provides higher-order functions to make code more efficient and readable.

* lru_cache → optimize recursion
* partial() → fix function arguments

### 2. LRU Cache

### 3. Partial
