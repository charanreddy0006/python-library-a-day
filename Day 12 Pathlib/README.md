# Day 13 - Pathlib Module

## 📌 Overview

On Day 13, I explored Python’s **pathlib module**, which provides an easy and modern way to work with file paths and directories.

It is cleaner and more readable compared to the os module.

---

## 🧠 Topics Covered

* Path.cwd() → current directory
* mkdir() → create folder
* write_text() → create/write file
* read_text() → read file
* exists() → check file
* iterdir() → list files
* unlink() → delete file

---

## 💻 Code Explanation

### 1. Create Folder

```python
Path("folder").mkdir()
```

### 2. Create File

```python
file.write_text("data")
```

### 3. Read File

```python
file.read_text()
```
