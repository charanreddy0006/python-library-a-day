# Day 11 - Sys Module

## 📌 Overview

On Day 11, I explored Python’s built-in **sys module**, which provides access to system-specific parameters and functions.

It is commonly used for command-line applications.

---

## 🧠 Topics Covered

* sys.argv → command-line arguments
* sys.version → Python version
* sys.exit() → exit program

---

## 💻 Code Explanation

### 2. Python Version
