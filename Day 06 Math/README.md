# Day 06 - Math Module

## 📌 Overview

On Day 06, I explored Python’s built-in **math module**, which provides mathematical functions and constants.

It is widely used in scientific calculations, data analysis, and problem solving.

---

## 🧠 Topics Covered

* Square root and power
* Rounding (ceil, floor)
* Constants (pi, e)
* Factorial
* Trigonometric functions
* Logarithmic functions

---

## 💻 Code Explanation

### 1. Square Root

```python
math.sqrt(25)
```

### 2. Power

```python
math.pow(2, 3)
```

### 3. Factorial

```python
math.factorial(5)
```

### 4. Trigonometry

```python
math.sin(math.radians(90))
```
