# Day 13 - Itertools Module

## 📌 Overview

On Day 13, I explored Python’s built-in **itertools module**, which provides fast and memory-efficient tools for working with iterators.

---

## 🧠 Topics Covered

* permutations() → all arrangements
* combinations() → selection without order
* combinations_with_replacement()
* product() → cartesian product
* count() → infinite sequence
* cycle() → repeat elements

---

## 💻 Code Explanation

### 1. Permutations

```python
itertools.permutations(data)
```

### 2. Combinations

```python
itertools.combinations(data, 2)
```

### 3. Product

```python
itertools.product(data, repeat=2)
```
