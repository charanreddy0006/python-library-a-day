# Day 10 - Collections Module

## 📌 Overview

On Day 10, I explored Python’s **collections module**, which provides advanced data structures beyond built-in lists and dictionaries.

It makes code more efficient and readable.

---

## 🧠 Topics Covered

* Counter → count elements
* defaultdict → default values
* namedtuple → structured data
* deque → fast insert/remove

