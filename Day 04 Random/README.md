# Day 04 - Python Random Module

## 📌 Overview

On Day 04, I explored Python’s built-in **random module**, which is used to generate random values and perform random operations.

It is useful in games, simulations, testing, and real-world applications.

---

## 🧠 Topics Covered

* random() → random float between 0 and 1
* randint() → random integer
* randrange() → random number with step
* choice() → random item from list
* sample() → multiple unique items
* shuffle() → shuffle list
* uniform() → random float in range

---

## 💻 Code Explanation

### 1. Random Integer

```python
random.randint(1, 10)
```

### 2. Random Choice

```python
random.choice(list)
```

### 3. Shuffle List

```python
random.shuffle(list)
```

### 4. Sampling

```python
random.sample(list, k)
```
